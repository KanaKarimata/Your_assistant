## サイト概要
### サイトテーマ：会社の秘書が役員のスケジュールを楽に管理できるアプリです。

### テーマを選んだ理由：前職で秘書業務を行っていたときに、役員のスケジュール管理が簡単にできるアプリが欲しいな、と思ったことからこのテーマにしました。


### ターゲットユーザ：役員及び秘書

### 主な利用シーン
【秘書側】役員に新しい予定が追加される際に、どの日時が空いているかの確認と予定の追加ができる
【役員側】スケジュールの確認ができ、予定変更の通知を受ける
